Rice Disease Detection Project

Overview

The Rice Disease Detection Project aims to develop an automated system for the early detection of diseases in rice plants. By leveraging convolutional neural networks (CNNs) and machine learning algorithms, the system will assist farmers in identifying and managing rice diseases effectively, leading to improved crop yield and agricultural sustainability.

CNN Models

Two CNN models were developed for rice disease detection: a lightweight model with four classes and a more complex model with nine classes. The second model was trained using transfer learning technique, specifically DenseNet121 architecture, to achieve high accuracy and robustness in disease classification whilst the first light weight model was architectured from ground-up meaning no transfer learning was used.

Theory about CNN

Convolutional neural networks (CNNs) are a class of deep learning models widely used in computer vision tasks, including image recognition and classification. CNNs consist of multiple layers, such as convolutional layers, pooling layers, and fully connected layers, which extract hierarchical features from input images and enable accurate disease classification.

End Product Web App

The project culminates in a web application implemented using Flask framework. The web app provides an intuitive user interface for uploading rice plant images, selecting CNN models for disease prediction, and viewing the predicted results in real-time. The web app is designed to be accessible and user-friendly for both farmers and agricultural professionals.

User Login

User authentication functionality is integrated into the web application, allowing users to create accounts, securely log in, and access personalized features. User login ensures data privacy and security, enabling personalized user experiences and facilitating user management for administrators.

Image Upload and Prediction

The web application enables users to upload images of rice plants and select CNN models for disease prediction. Upon uploading an image, the selected CNN model analyzes the image and predicts the presence of any diseases. The predicted results are displayed to the user, aiding in early disease detection and crop management.
